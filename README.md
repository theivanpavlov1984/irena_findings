# Irena | Находки

Витрина персонального байер-сервиса: премиальные сумки и украшения под заказ
с доставкой и проверкой подлинности. Лицо сервиса — Ирина; закупка идёт с
азиатского ресейла (Япония и Гонконг), проверка — Entrupy (сумки) и доверенный
ювелир (украшения).

Фронтенд — одностраничное приложение на React. Бэкенда пока нет: каталог хранится
прямо в компоненте, бронь уходит в личные сообщения Telegram.

## Стек

- **React 18** — UI
- **Vite 5** — сборка и dev-сервер
- **lucide-react** — иконки

Стили — инлайн + один блок `<style>` внутри компонента (без Tailwind/CSS-файлов).
Все шрифты, фото и логотипы зашиты в код в base64, поэтому `src/IrenaStore.jsx`
весит много — это норма, внешних ассетов нет.

## Запуск

```bash
npm install      # поставить зависимости
npm run dev      # дев-сервер (обычно http://localhost:5173)
npm run build    # прод-сборка в dist/
npm run preview  # посмотреть собранную версию
```

## Структура

```
.
├── index.html            # точка входа, монтирует #root
├── vite.config.js        # конфиг Vite (плагин react)
├── package.json
└── src
    ├── main.jsx          # рендер <App/> в #root
    └── IrenaStore.jsx    # всё приложение (default export App)
```

## Что внутри

- Главная: герой-карусель, новинки, блок коллекций, баннер «Подлинность».
- Каталог: фильтры (бренд, коллекция, категория, состояние, металл, цена),
  пагинация, две вкладки — сумки и украшения.
- Страница товара: хлебные крошки, галерея (1 большое фото + миниатюры),
  характеристики, блок проверки подлинности, «Вам может понравиться».
- Форма заявки на поиск под заказ, страница «Подлинность», избранное.
